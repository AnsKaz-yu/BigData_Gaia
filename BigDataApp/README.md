
# Gaia Data Analyzer

**Gaia Data Analyzer** — это графическое приложение для анализа и визуализации астрономических данных из каталога Gaia (DR3). Программа позволяет загружать, предварительно обрабатывать и анализировать большие массивы данных, а также строить графики по различным задачам.

## Возможности

- Загрузка и разбиение данных Gaia DR3 на фрагменты
- Предварительная обработка данных для различных задач
- Анализ и визуализация результатов по 6 задачам (Task1–Task6)
- Удобный графический интерфейс на базе Tkinter
- Сохранение настроек рабочей директории

## Требования

- Python 3.10 или выше (рекомендуется 3.10–3.12)
- pip (менеджер пакетов Python)
- Операционная система: Windows, Linux, macOS

## Установка

1. **Клонируйте репозиторий или скачайте архив с исходным кодом.**
2. **Перейдите в корневую папку проекта:**
3. **Установите зависимости и само приложение:**
	```
	pip install ./application
	```
	Это установит все необходимые библиотеки (astropy, astroquery, numpy, matplotlib, pandas и др.) и создаст консольную команду `gaia_analyze`.

## Запуск приложения

После установки выполните команду:
```
gaia_analyze
```
Откроется графический интерфейс приложения.

## Быстрый старт

1. **Выберите рабочую директорию** — при первом запуске появится предупреждение. Укажите папку, где будут храниться загруженные и обработанные данные.
2. **Загрузка данных** — во вкладке "Загрузка данных" выберите и загрузите нужные фрагменты каталога Gaia.
3. **Предварительная обработка** — во вкладке "Предварительная обработка данных" обработайте загруженные фрагменты для выбранной задачи.
4. **Анализ и визуализация** — во вкладке "Анализ данных и визуализация" выберите обработанные фрагменты и постройте графики.

## Примечания

- Для корректной работы требуется стабильное интернет-соединение (для загрузки данных из Gaia).
- Все настройки (рабочая директория) сохраняются в файл `settings.txt` в папке запуска.
- Если при запуске возникают ошибки с импортами, убедитесь, что пакет установлен через pip, а не просто запускается из исходников.
